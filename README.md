# Sistemas Operativos

## Qué es un Sistema Operativo

Un sistema operativo es el conjunto de programas que gestiona los recursos del hardware y provee servicios a los programas de aplicación de software.

## Ejemplos de Sistema Operativos

-PC:
  - Windows
  - GNU/Linux:
    - Ubuntu.
    - Mint.
    - Arch.
    - Debian.
    - Fedora.
  - Mac
- Dispositivos móviles:
  - Android
  - iOS
